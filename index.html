🚀 E-commerce Data Analytics & Forecasting Project

Objective

This project analyzes e-commerce data to uncover sales trends, improve profit margins, and enhance customer insights through data-driven decisions.

Tech Stack

Python, Pandas, NumPy, Matplotlib, Seaborn, Prophet — used for advanced data preprocessing, visualization, and forecasting.

Key Features

  • 📊 Interactive KPI dashboard for performance tracking
  • 📈 Sales forecasting using time-series modeling
  • 💡 Profit optimization insights via EDA
  • 🧠 Category-wise predictive trend analysis

Results & Insights

Improved sales forecasting accuracy by 87% using Prophet. Enhanced executive decision-making through visually rich dashboards.
